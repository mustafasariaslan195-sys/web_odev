📋 To-Do List Web Uygulaması Bu proje, kullanıcı kayıt ve giriş sistemi bulunan basit bir To-Do List (Görev Yönetim) web uygulamasıdır. Kullanıcılar sisteme kayıt olabilir, giriş yapabilir ve yalnızca kendilerine ait görevleri ekleyip silebilirler. Proje; Frontend, Backend ve Veritabanı entegrasyonunu öğrenmek isteyenler için hazırlanmış örnek bir web uygulamasıdır. 🎯 Projenin Amacı Bu proje aşağıdaki web geliştirme konularını öğretmeyi amaçlamaktadır: 👤 Kullanıcı kayıt sistemi oluşturma 🔐 Kullanıcı giriş sistemi geliştirme 🗄️ PHP ile MySQL veritabanı bağlantısı kurma 🔑 Session yönetimi kullanımı 📝 CRUD işlemleri Create (Ekleme) Read (Listeleme) Delete (Silme) 👥 Kullanıcıya özel veri listeleme mantığı 🔒 Şifrelerin güvenli şekilde hashlenmesi 🎨 Basit form ve kullanıcı arayüzü tasarımı 🛠️ Kullanılan Teknolojiler Projede aşağıdaki teknolojiler kullanılmıştır: HTML → Sayfa yapısı CSS → Basit arayüz tasarımı JavaScript → Form doğrulama (isteğe bağlı) PHP → Sunucu tarafı işlemler MySQL → Veritabanı yönetimi ⚙️ Sistem Nasıl Çalışır? Uygulamanın çalışma mantığı şu şekildedir: Kullanıcı sisteme kayıt olur. Kullanıcının şifresi hashlenerek veritabanına kaydedilir. Kullanıcı giriş yaptığında bir session oluşturulur. Kullanıcı yalnızca kendi görevlerini görüntüleyebilir. Kullanıcı yeni görev ekleyebilir. Kullanıcı görevlerini silebilir. Kullanıcı çıkış yaptığında session sonlandırılır. 📂 Uygulama Özellikleri Uygulamada bulunan temel özellikler: 👤 Kullanıcı kayıt sistemi 🔑 Güvenli kullanıcı giriş sistemi 📋 Kullanıcıya özel görev listesi ➕ Görev ekleme ❌ Görev silme 🚪 Güvenli çıkış (Logout) 🎓 Kimler İçin Uygun? Bu proje özellikle aşağıdaki kişiler için uygundur: Backend geliştirmeye yeni başlayanlar PHP & MySQL pratiği yapmak isteyenler Session mantığını öğrenmek isteyenler Küçük ölçekli web uygulaması geliştirmek isteyen öğrenciler 🚀 Projeyi Çalıştırma Projeyi çalıştırmak için aşağıdaki adımları takip edebilirsiniz: Repository'i klonlayın git clone https://github.com/mustafasariaslan195-sys/web_odev.git Projeyi XAMPP / WAMP / Localhost ortamına yerleştirin. MySQL üzerinde gerekli veritabanını oluşturun. Tarayıcıdan projeyi çalıştırın: http://localhost/web_odev
